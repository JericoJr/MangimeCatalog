App Description: Mangime is a personal digital library and journal website for browsing all anime- and manga-related content. This website will allow users to explore and browse different content like manga and anime via Jikan API, as well as record, store, and see their favorite content including the fields assigned to the content such as title, type, genre, status (completed, waitlist, reading/watching), personal rating, and their own comments on the content for future reference.
